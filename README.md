Barbería Pro
Barbería Pro es una aplicación de escritorio hecha con Python + Flet, diseñada para gestionar de manera sencilla clientes, barberos, servicios, turnos y ventas dentro de una barbería.
Su objetivo es ser rápida, intuitiva y funcional, sin complicaciones.
Características principales
Inicio de sesión con usuario administrador por defecto
Gestión completa de clientes
Módulo de barberos, servicios, turnos y ventas
Base de datos SQLite integrada
Interfaz moderna con barra lateral y vistas organizadas
